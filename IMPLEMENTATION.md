IMPLEMENTASI-PLAN.md: AI-Driven Data Academy (Rendy & Abdul)1. Executive Summary & ArchitectureTujuan: Membangun platform edukasi Data Analyst & Data Scientist berstandar internasional ("Zero to Hero") yang didukung oleh AI Instructors (Rendy & Abdul) dengan kemampuan penilaian kode otomatis, kurikulum adaptif, dan lingkungan coding interaktif.High-Level ArchitectureFrontend: Next.js (React) + TypeScript + Tailwind CSS (Untuk performa, SEO, dan UI modern).Backend API: FastAPI (Python) - Pilihan terbaik karena native dengan library data science.AI Engine (The Brain): LangChain + OpenAI GPT-4o / Claude 3.5 Sonnet (Agentic Workflow).Code Execution Environment (Sandbox): JupyterLite (Client-side) atau Docker Containers (Server-side) untuk menjalankan kode Python/SQL user secara aman.Database: PostgreSQL (User data, Curriculum) + Vector DB (Pinecone/Milvus untuk Knowledge Base materi).2. Persona & AI Behavior DesignSistem ini menggunakan System Prompts yang kompleks untuk menciptakan karakter dosen yang hidup.A. Rendy (Dosen Data Analyst)Fokus: Business Intelligence, SQL, Data Cleaning, Data Visualization (Tableau/PowerBI), Storytelling.Kepribadian: Pragmatis, berorientasi bisnis, tegas pada detail visualisasi, santai tapi teliti.Teaching Style: "Data is money." Selalu mengaitkan teknis dengan business decision.B. Abdul (Dosen Data Scientist)Fokus: Mathematics/Statistics, Machine Learning, Deep Learning, NLP, MLOps, Python Engineering.Kepribadian: Akademis, sangat logis, menekankan pada "Why" (matematika di balik algoritma), perfeksionis dalam efisiensi kode.Teaching Style: "Trust the math." Pendekatan eksperimental dan ilmiah.3. Technical Features & Implementation DetailA. Landing Page & OnboardingHero Section: Split screen design. Kiri (Rendy - Analyst) vs Kanan (Abdul - Scientist). Visual interaktif.Assessment Test: Sebelum mulai, user bisa mengambil tes singkat. AI akan merekomendasikan level awal (apakah benar-benar dari nol atau bisa skip level dasar).B. Interactive Classroom (LMS Interface)Ini adalah "Jantung" aplikasi. Tampilan terbagi menjadi 3 panel:Panel Materi (Kiri): Video/Teks interaktif + Chat dengan Dosen AI (Rendy/Abdul).Panel Code Editor/Workspace (Tengah): VS Code embedded (Monaco Editor) yang mendukung SQL dan Python.Panel Output & Feedback (Kanan): Hasil run code + AI Grader.C. The "AI Grader" & Scoring System (Poin 5)Sistem penilaian tidak manual. AI akan menilai kode berdasarkan 4 metrik:Correctness (40%): Apakah output sesuai kunci jawaban?Efficiency (20%): (Khusus Abdul) Apakah kode efisien (Big O notation)?Style (20%): Apakah kode bersih dan mengikuti standar (PEP8)?Business Insight (20%): (Khusus Rendy) Apakah kesimpulan data benar secara bisnis?Contoh Feedback Otomatis (Persona Rendy):"Halo! Query SQL kamu hasilnya benar, tapi secara performa ini lambat karena kamu menggunakan terlalu banyak Sub-query. Coba gunakan Common Table Expression (CTE) agar lebih rapi dan cepat. Ingat, di industri, waktu query adalah uang!"D. Rekomendasi & Adaptive Learning (Poin 6)Setelah menyelesaikan level/quiz:Jika skor < 70: AI menyarankan materi spesifik untuk diulang (bukan mengulang semua).Jika skor > 90: AI memberikan "Bonus Challenge" (Materi pengayaan tingkat lanjut).5.
